## 先说 AxMath 是什么

AxMath 是一款国产数学公式编辑器，由一位浙大老师开发，用来替代 MathType 的活儿。核心功能就一件事：帮你把复杂的数学公式快速打进 Word 或 WPS，而且效果好看、不变形。

安装完之后，它会以插件形式嵌入你的 Office 或 WPS 菜单栏，点击就能调出编辑窗口，所见即所得，不用记 LaTeX 语法也能用——当然，它也支持 LaTeX 代码输入，双向转换，学 LaTeX 的同学也不亏。

功能清单大概是这样的：

- **公式编辑**：点选模板、快捷键、脚本三种输入方式，支持手写识别
- **LaTeX 双向转换**：Ctrl+L 打开 LaTeX 输入窗口，公式和代码互相转
- **科学计算**：不只是排版，还能真的算——加减乘除、开根、幂、三角函数都支持，能定义变量和常数
- **OLE 嵌入**：无缝插入 Word / WPS，行内公式还支持与正文基线对齐
- **图片输出**：BMP、JPG、PNG、GIF、矢量图格式全支持
- **公式库管理**：支持公式编号、公式库存储，多文档间可以直接调用
